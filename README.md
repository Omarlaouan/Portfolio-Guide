# Machine Learning and Deep Learning

| Project Link | Tools | Algorithms / Methods | Project Description |
|---|---|---|---|
| 🧬 [MedMLOps Treatment Response Platform](https://github.com/Omarlaouan/medmlops-treatment-response-platform) | Python, scikit-learn, FastAPI, Streamlit, MLflow, Docker, GitHub Actions, Pytest, Ruff | Explainable ML, Data Validation, Drift Monitoring, MLOps, Model Serving | A mini MLOps platform for explainable treatment-response prediction in healthcare. The project simulates a healthcare ML pipeline for antimicrobial resistance and antibiotic susceptibility prediction using synthetic AMR-like data. It covers data generation, schema validation, cohort construction, feature engineering, model training, evaluation, inference, explainability, monitoring, FastAPI serving, Streamlit demo, Docker packaging, CI, and model governance documentation. This is an educational prototype and is not intended for clinical decision-making. |
| 📄 [Citation Link Prediction](https://github.com/Omarlaouan/citation_prediciton) | Python, Jupyter Notebook, Gensim, NetworkX | Graph Neural Network, Link Prediction, NLP, Graph ML | The goal of this project is to apply machine learning and artificial intelligence techniques to citation link prediction: determining whether one research paper cites another. The citation network contains several thousand research papers, along with their abstracts and authors. The project uses graph structure and textual information to train a classifier that predicts whether two nodes should be connected by an edge. [Detailed project presentation](https://github.com/Omarlaouan/citation_prediciton/blob/main/Project%20presentation%20-%20citation%20prediction.pdf). |
| 💰 [Credit Default Prediction](https://github.com/Omarlaouan/credit_default_prediction) | Python, Jupyter Notebook | Decision Trees, Supervised Learning, Classification | This machine learning project aims to predict whether a customer will default on a loan within 60 days of disbursement. The project uses historical customer financial data to train, compare, and evaluate several machine learning models, with the objective of selecting the best-performing model for deployment. [Detailed project presentation](https://github.com/Omarlaouan/credit_default_prediction/blob/main/Project%20presentation.pdf). |
| 🚗 [Automatic Carbon Footprint Calculator](https://github.com/Omarlaouan/Automatic-carbon-footprint-calculator) | Python | Tree-Based Models, Time Series, Computer Vision, NLP | This repository explores how machine learning can be used to automate carbon footprint calculation across different use cases. The project builds on the methodology and findings presented in my thesis: [AI and Carbon Footprint — Omar LAOUAN](https://github.com/Omarlaouan/Automatic-carbon-footprint-calculator/blob/main/IA%20et%20empreinte%20carbonne_Omar%20LAOUAN.pdf).
